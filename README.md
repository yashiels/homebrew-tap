# yashiels/tap

Homebrew tap for CLI tools and macOS apps by [Yashiel Sookdeo](https://github.com/yashiels).

## Tap

```bash
brew tap yashiels/tap
```

## Packages

| Formula | Description |
|---------|-------------|
| `checkers60` | Shop Checkers Sixty60 from the terminal |
| `takealot` | Search, cart, and checkout on Takealot without a browser |
| `postnet` | Track PostNet parcels in one command |
| `woolies` | Woolworths Dash grocery delivery from your shell |
| `orro` | Control your Tuya standing desk from the terminal |
| `lnk` | LinkedIn CLI — search jobs, view profiles, apply from the terminal |
| `twt` | X/Twitter from the terminal |

| Cask | Description |
|------|-------------|
| `keepawake` | Menu bar app to prevent Jamf-managed macOS screen lock |

## Install

```bash
brew install yashiels/tap/<formula>          # formulas
brew install --cask yashiels/tap/<cask>      # casks
```

## Update

```bash
brew update && brew upgrade
```

## Uninstall

```bash
brew uninstall <formula>
brew untap yashiels/tap   # optional — removes the tap entirely
```
