# Homebrew Tap

Homebrew tap for tools by [Yashiel Sookdeo](https://github.com/yashiels).

## Install

```bash
brew install --cask yashiels/tap/keepawake
```

## Available Casks

| Cask | Description |
|------|-------------|
| [keepawake](Casks/keepawake.rb) | Menu bar app to prevent Jamf-managed macOS screen lock |
